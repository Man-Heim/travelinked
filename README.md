# TraveLinked

This was a student project developed during the Bachelor studys of Computer Science and Media and presented at the Media Night in the summer term 2021, more Information can be found under the Project Link.

Project Link: https://www.hdm-stuttgart.de/stage/projekt_detail/projekt_details?projekt_ID=3595

![image](https://github.com/Man-Heim/travelinked/assets/64532251/da72fd77-3946-4873-af2b-11ad3e223453)

TraveLinked ist eine Webanwendung für die individuelle Reiseplanung mit Fokus auf soziale Interaktionen. Mit einem persönlichen Benutzeraccount, kann man eigene Touren anlegen und diese mit anderen Benutzern teilen. Diese Touren können von anderen Reisenden bewertet werden. Durch die große Vernetzung innerhalb der Community, hat man die Möglichkeit als Local Guide zu agieren, um anderen Benutzern Geheimtipps zu empfehlen!

TraveLinked is a web application for individual travel planning with a focus on social interaction. With a personal user account, you can create your own tours and share them with other users. These tours can be rated by other travelers. Thanks to the large network within the community, you have the opportunity to act as a local guide and recommend insider tips to other users!

The project is currently not running anymore!
